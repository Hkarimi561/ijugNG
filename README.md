# Iranian Java User Group Website

## Code with :

* [Angular 18](https://angular.dev/)
* [Tailwind 3.4](https://tailwindcss.com/)
* [daisyui 4.12](https://daisyui.com/)

## How to Start:

after Clone the project you need run :

```bash
npm install
```

For development:

```bash
ng serve
```

and for build:

```bash
ng build
```
